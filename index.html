VAT Calculation Rules
SA SARS Standard · 15% rate
Extract Excl. from Incl.
excl = incl / 1.15
e.g. R1,000 incl. → R869.57 excl.
Extract VAT from Incl.
vat = incl - (incl / 1.15)
e.g. R1,000 incl. → R130.43 VAT
Calculate Incl. from Excl.
incl = excl * 1.15
e.g. R869.57 excl. → R1,000 incl.
